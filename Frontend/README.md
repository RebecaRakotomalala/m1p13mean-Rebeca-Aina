MallConnect is a free Angular admin dashboard template built with Angular and the latest Bootstrap 5. It is meant to provide the best possible User Experience with highly customizable, feature-rich pages. It is a comprehensive dashboard template featuring an easy-to-use and intuitive responsive design, suitable for viewing on both retina screens and laptops.

### Name Derived From Nature

The MallConnect Logo is inspired by the insect name 'MallConnectes', as they have triangular heads with flexible Necks. Also, the name is derived from two popular design systems, Material and Ant Design (M-Ant-is).

## Getting Started

1. Clone from GitHub

```
git clone https://github.com/codedthemes/mallconnect-free-angular-admin-template.git
```

2. Install packages

```
yarn
```

3. Run project

```
yarn start
```

## Why MallConnect? 

MallConnect offers everything you need to create dashboards. We have included the following high-end features in our initial release:

- Modern aesthetics UI design
- Bootstrap components
- Fully Responsive, all modern browsers supported
- Easy to use code structure
